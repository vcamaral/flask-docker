# flask-docker

Exemplo de aplicação [Flask](http://flask.pocoo.org) que consome dados de um [Redis](https://redis.io) e roda em um container [Docker](https://www.docker.com).

## Como executar:

```
docker build -t flask-docker .
docker-compose up -d
```
